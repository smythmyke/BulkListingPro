and sub-niche, all within Etsy's 20-character tag limit.


HOW TO USE - QUICK START:
-------------------------

  1. Pick the niche spreadsheet that fits your shop

  2. Open it in Excel or Google Sheets

  3. Add your own product images in the image_1 through image_5 columns
     (use full file paths like C:\Users\YourName\Images\product.jpg)

  4. Add your digital download file in the digital_file_1 column

  5. Edit any titles, descriptions, prices, or tags to match your brand

  6. Save the spreadsheet


BULK UPLOAD WITH BULKLISTINGPRO:
--------------------------------

  Instead of creating each listing by hand on Etsy, use BulkListingPro
  to upload all 20 listings automatically:

  1. Install BulkListingPro from the Chrome Web Store
     Search "BulkListingPro" at: https://chromewebstore.google.com

  2. Launch Chrome with remote debugging enabled:
     Close all Chrome windows, then run:
     chrome.exe --remote-debugging-port=9222

  3. Log into your Etsy shop in the browser

  4. Open the BulkListingPro sidepanel (click the extension icon)

  5. Sign in with Google and ensure you have credits

  6. Drag and drop your niche spreadsheet into BulkListingPro

  7. Preview your listings in the queue

  8. Click "Start Upload" - BulkListingPro creates all listings on Etsy

  All listings are saved as DRAFTS so you can review them on Etsy
  before publishing.
